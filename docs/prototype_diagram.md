## 🔄 交互流程原型图

### 角色切换交互流程
```
初始状态:
[李总监] [王专家] [张顾问] [刘专员]
  ████    ░░░░    ░░░░    ░░░░   (当前角色高亮)

用户点击"王专家":
[李总监] [王专家] [张顾问] [刘专员]
  ░░░░    ████    ░░░░    ░░░░

界面变化:
1. 头像切换: 👤 → 🛠️
2. 角色标识: "高级客户经理" → "产品专家"
3. 可用工具更新
4. 对话风格调整
```

```
用户点击"📊分析"工具:

1. 工具面板展开:
   ┌─────────────────┐
   │     分析工具     │
   │  □ 数据统计     │
   │  □ 趋势分析     │
   │  □ 对比报告     │
   │  [执行分析]      │
   └─────────────────┘

2. 用户选择选项并确认
3. 智能体响应:
   🤖 李总监: 🔧 正在执行数据分析...
   💭 思维链: 获取数据 → 清洗处理 → 统计分析 → 生成报告
4. 结果显示在对话区域
```

## 🎯 关键界面元素详细设计

### 1. 角色卡片设计规范
```
┌─────────────────────────┐
│       角色卡片           │
│  ┌─────┐                │
│  │ 👤  │ 李总监         │
│  │●在线│ 高级客户经理    │
│  └─────┘                │
│  战略规划 · 客户关系     │
│  [切换角色]              │
└─────────────────────────┘

设计要素:
- 头像区域: 64x64px圆形
- 状态指示: ●在线 ○忙碌 ◌离线
- 专长标签: 最多显示2个主要专长
- 切换按钮: 突出显示当前角色
```

### 2. 消息气泡设计规范
```
用户消息气泡:
┌──────────────────────────────┐
│ 👤 用户                      │
│ 我们需要一个企业云迁移方案    │
│ 10:30 AM                    │
└──────────────────────────────┘

智能体消息气泡:
┌──────────────────────────────┐
│ 🤖 李总监 · 高级客户经理      │
│ 💭 思维链展开                │
│ 好的，我来为您制定方案...     │
│ 🔧 调用了: 方案生成工具       │
│ 10:31 AM                    │
└──────────────────────────────┘

设计要点:
- 头像标识: 区分用户和智能体
- 角色标签: 显示当前对话角色
- 思维链指示: 可展开的思考过程
- 工具调用标记: 显示使用的工具
- 时间戳: 精确到分钟
```

### 3. 工具面板设计规范
```
工具分类面板:
┌─────────────────────────┐
│ 📁 文件操作             │
│ ├─ 📖 读取文件          │
│ ├─ ✏️ 编辑文档          │
│ └─ 💾 保存文件          │
│                         │
│ 📊 数据分析             │
│ ├─ 🧮 计算器           │
│ ├─ 📈 图表生成          │
│ └─ 🔢 统计分析          │
│                         │
│ [收起面板]               │
└─────────────────────────┘

交互逻辑:
- 分类折叠: 可展开/收起每个工具分类
- 工具状态: 可用/禁用状态视觉区分
- 权限控制: 根据角色显示可用工具
- 快捷操作: 常用工具置顶显示
```

## 📱 响应式断点设计

### 断点定义
```css
/* 移动端优先设计 */
@media (min-width: 640px) { /* sm */ }
@media (min-width: 768px) { /* md */ }
@media (min-width: 1024px) { /* lg */ }
@media (min-width: 1280px) { /* xl */ }
```

### 布局变化
- **< 768px**: 单列布局，角色条水平滚动
- **768px - 1024px**: 侧边栏固定宽度，主区域自适应
- **> 1024px**: 完整三栏布局，工具面板常驻显示
